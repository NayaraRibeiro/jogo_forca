# Jogo da Forca

O objetivo do jogo é adivinhar a palavra aleatória sorteada pela computador.

## Pré-requisitos

Para rodar o jogo é necessário ter Ruby instalado em seu pc.

## Executar

Basta rodar o arquivo main.rb no terminal:
ruby main.rb

## Ferramentas utilizadas

* Ruby
